# Formula 1 Winners Analysis

A simple data analysis project using Python and SQL to explore 
Formula 1 race winners from 1950 to 2025.

## What this project does

- Loads a dataset with 1,142 F1 races
- Stores the data in a local SQLite database
- Queries the top 10 most winning drivers of all time
- Visualizes the result in a bar chart

## Technologies used

- Python
- Pandas
- SQLite3
- Matplotlib

## How to run

pip install -r requirements.txt
python main.py

## Dataset

Downloaded from Kaggle — Formula 1 World Championship results (1950 to mid-2025).
